Emphasize text with color highlights or underlines.

## Highlight

Mark text with a colored background.

```mdx
<Highlight>important text</Highlight>
<Highlight color="green">success</Highlight>
```

<Highlight>important text</Highlight>
<Highlight color="green">success</Highlight>
<Highlight color="blue">info</Highlight>
<Highlight color="red">error</Highlight>
<Highlight color="purple">special</Highlight>

## Highlight colors

### Yellow

Default highlight color.

<Highlight>default yellow</Highlight>

### Green

For positive or success text.

<Highlight color="green">completed</Highlight>

### Blue

For informational text.

<Highlight color="blue">note this</Highlight>

### Red

For warnings or errors.

<Highlight color="red">critical</Highlight>

### Purple

For special emphasis.

<Highlight color="purple">featured</Highlight>

## Underline

Add styled underlines to text.

```mdx
<Underline>underlined text</Underline>
<Underline style="wavy">wavy</Underline>
```

<Underline>solid</Underline> <Underline style="wavy">wavy</Underline>
<Underline style="dotted">dotted</Underline>
<Underline style="dashed">dashed</Underline>

## Underline styles

### Solid

Default underline style.

<Underline>solid underline</Underline>

### Wavy

Wavy underline for emphasis.

<Underline style="wavy">wavy underline</Underline>

### Dotted

Dotted underline.

<Underline style="dotted">dotted underline</Underline>

### Dashed

Dashed underline.

<Underline style="dashed">dashed underline</Underline>

## Props

### Highlight

| prop     | type                                               | default  | description       |
| -------- | -------------------------------------------------- | -------- | ----------------- |
| children | ReactNode                                          | required | Text to highlight |
| color    | "yellow" \| "green" \| "blue" \| "red" \| "purple" | "yellow" | Highlight color   |

### Underline

| prop     | type                                      | default  | description       |
| -------- | ----------------------------------------- | -------- | ----------------- |
| children | ReactNode                                 | required | Text to underline |
| style    | "solid" \| "wavy" \| "dotted" \| "dashed" | "solid"  | Underline style   |
