Displays estimated reading time for content based on word count.

## Usage

```tsx
import { ReadTime } from "fromsrc";

<ReadTime content={doc.content} />;
```

## Props

| prop      | type   | default  | description                         |
| --------- | ------ | -------- | ----------------------------------- |
| content   | string | required | Text content to calculate time from |
| wpm       | number | 200      | Words per minute reading speed      |
| className | string | -        | Additional CSS classes              |

## Example

```tsx
<ReadTime content={doc.content} wpm={250} />
```

Renders as "3 min read"

## Utility function

You can also use the utility function directly:

```tsx
import { calcReadTime } from "fromsrc";

const minutes = calcReadTime(content);
const minutes = calcReadTime(content, 250); // custom wpm
```
