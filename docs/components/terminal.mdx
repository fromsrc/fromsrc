display command line output with syntax highlighting.

## usage

```mdx
<Terminal>
  <Line>npm install fromsrc</Line>
  <Output>added 42 packages in 2s</Output>
</Terminal>
```

<Terminal>
  <Line>npm install fromsrc</Line>
  <Output>added 42 packages in 2s</Output>
</Terminal>

## custom title

```mdx
<Terminal title="bash">
  <Line>git status</Line>
  <Output>On branch main</Output>
</Terminal>
```

<Terminal title="bash">
  <Line>git status</Line>
  <Output>On branch main</Output>
</Terminal>

## custom prompt

```mdx
<Terminal>
  <Line prompt=">">.exit</Line>
</Terminal>
```

<Terminal>
  <Line prompt=">">.exit</Line>
</Terminal>

## typed animation

animate text appearing character by character:

```mdx
<Terminal>
  <Line><Typed text="npx create-fromsrc my-docs" speed={30} /></Line>
</Terminal>
```

<Terminal>
  <Line><Typed text="npx create-fromsrc my-docs" speed={30} /></Line>
</Terminal>

## props

### terminal

| prop | type | default | description |
|------|------|---------|-------------|
| title | string | "Terminal" | window title |
| children | ReactNode | - | content |

### line

| prop | type | default | description |
|------|------|---------|-------------|
| prompt | string | "$" | prompt character |
| children | ReactNode | - | command text |

### output

| prop | type | default | description |
|------|------|---------|-------------|
| children | ReactNode | - | output text |

### typed

| prop | type | default | description |
|------|------|---------|-------------|
| text | string | - | text to animate |
| speed | number | 50 | ms per character |
