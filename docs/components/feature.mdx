highlight features with icons and descriptions.

## Feature

inline feature with icon:

```mdx
<Feature icon="⚡" title="fast builds">
  incremental compilation for instant feedback
</Feature>
```

<Feature icon="⚡" title="fast builds">
  incremental compilation for instant feedback
</Feature>

## Features grid

multiple features in a grid:

```mdx
<Features columns={2}>
  <FeatureCard icon="🔍" title="search">
    full-text search across all docs
  </FeatureCard>
  <FeatureCard icon="🎨" title="themes">
    dark and light mode support
  </FeatureCard>
</Features>
```
