show project structure with folders and files.

## usage

```mdx
<Files>
  <Folder name="app" defaultOpen>
    <File name="page.tsx" />
    <File name="layout.tsx" />
  </Folder>
  <Folder name="docs">
    <File name="index.mdx" />
  </Folder>
  <File name="package.json" />
</Files>
```

<Files>
  <Folder name="app" defaultOpen>
    <File name="page.tsx" />
    <File name="layout.tsx" />
  </Folder>
  <Folder name="docs">
    <File name="index.mdx" />
  </Folder>
  <File name="package.json" />
</Files>

## nested folders

folders can contain other folders:

```mdx
<Files>
  <Folder name="src" defaultOpen>
    <Folder name="components" defaultOpen>
      <File name="button.tsx" />
    </Folder>
  </Folder>
</Files>
```

<Files>
  <Folder name="src" defaultOpen>
    <Folder name="components" defaultOpen>
      <File name="button.tsx" />
    </Folder>
  </Folder>
</Files>

## with icons

files can have custom icons:

```mdx
<Files>
  <Folder name="config" defaultOpen>
    <File name="settings.json" icon={<span>&#9881;</span>} />
  </Folder>
</Files>
```

<Files>
  <Folder name="config" defaultOpen>
    <File name="settings.json" icon={<span>&#9881;</span>} />
  </Folder>
</Files>

## props

### folder

| prop | type | default | description |
|------|------|---------|-------------|
| name | string | - | folder name |
| defaultOpen | boolean | false | start expanded |

### file

| prop | type | description |
|------|------|-------------|
| name | string | file name |
| icon | ReactNode | custom icon |
