display additional information on hover.

## Hoverinfo

show type or value hints on hover.

```mdx
<HoverInfo info="string">name</HoverInfo>
<HoverInfo info="42" type="value">
  count
</HoverInfo>
```

hover over: <HoverInfo info="string">name</HoverInfo> or <HoverInfo info="42" type="value">count</HoverInfo> or <HoverInfo info="undefined" type="error">missing</HoverInfo>

## Types

### Type

default style for type annotations.

<HoverInfo info="number">age</HoverInfo>

### Value

for displaying values.

<HoverInfo info="true" type="value">
  enabled
</HoverInfo>

### Error

for error states.

<HoverInfo info="null reference" type="error">
  data
</HoverInfo>

## Typepopup

for richer hover content with custom formatting.

```mdx
<TypePopup
  trigger="config"
  content={
    <>
      type: object
      <br />
      keys: 3
    </>
  }
/>
```

<TypePopup
  trigger="config"
  content={
    <>
      type: object
      <br />
      keys: 3
    </>
  }
/>

## Props

### Hoverinfo

| prop     | type                         | default  | description     |
| -------- | ---------------------------- | -------- | --------------- |
| children | ReactNode                    | required | trigger element |
| info     | string                       | required | text to display |
| type     | "type" \| "value" \| "error" | "type"   | color style     |

### Typepopup

| prop    | type      | default  | description   |
| ------- | --------- | -------- | ------------- |
| trigger | ReactNode | required | hover trigger |
| content | ReactNode | required | popup content |
