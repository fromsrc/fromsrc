Display tabular data with optional sorting, custom rendering, and mobile responsiveness.

## Usage

```tsx
const data = [
  { name: "react", stars: 220000, language: "javascript" },
  { name: "vue", stars: 205000, language: "typescript" },
  { name: "svelte", stars: 75000, language: "typescript" },
]

const columns = [
  { key: "name", header: "name" },
  { key: "stars", header: "stars" },
  { key: "language", header: "language" },
]

<Table data={data} columns={columns} />
```

<Table
  data={[
    { name: "react", stars: 220000, language: "javascript" },
    { name: "vue", stars: 205000, language: "typescript" },
    { name: "svelte", stars: 75000, language: "typescript" },
  ]}
  columns={[
    { key: "name", header: "name" },
    { key: "stars", header: "stars" },
    { key: "language", header: "language" },
  ]}
/>

## Sorting

Enable sorting on columns with `sortable`:

```tsx
const columns = [
  { key: "name", header: "name", sortable: true },
  { key: "stars", header: "stars", sortable: true },
  { key: "language", header: "language" },
]

<Table data={data} columns={columns} defaultSort="stars" defaultDirection="desc" />
```

<Table
  data={[
    { name: "react", stars: 220000, language: "javascript" },
    { name: "vue", stars: 205000, language: "typescript" },
    { name: "svelte", stars: 75000, language: "typescript" },
  ]}
  columns={[
    { key: "name", header: "name", sortable: true },
    { key: "stars", header: "stars", sortable: true },
    { key: "language", header: "language" },
  ]}
  defaultSort="stars"
  defaultDirection="desc"
/>

## Custom rendering

use the `render` function to customize cell output:

```tsx
const columns = [
  { key: "name", header: "name" },
  {
    key: "stars",
    header: "stars",
    render: (value) => `${(value / 1000).toFixed(0)}k`,
  },
  {
    key: "language",
    header: "language",
    render: (value) => <Badge>{value}</Badge>,
  },
];
```

## Responsive

Hide columns on mobile with `hideOnMobile`:

```tsx
const columns = [
  { key: "name", header: "name" },
  { key: "stars", header: "stars" },
  { key: "language", header: "language", hideOnMobile: true },
  { key: "license", header: "license", hideOnMobile: true },
];
```

<Table
  data={[
    { name: "react", stars: 220000, language: "javascript", license: "mit" },
    { name: "vue", stars: 205000, language: "typescript", license: "mit" },
    { name: "svelte", stars: 75000, language: "typescript", license: "mit" },
  ]}
  columns={[
    { key: "name", header: "name" },
    { key: "stars", header: "stars" },
    { key: "language", header: "language", hideOnMobile: true },
    { key: "license", header: "license", hideOnMobile: true },
  ]}
/>

## Styles

### Striped

Alternating row colors:

```tsx
<Table data={data} columns={columns} striped />
```

<Table

    { name: "react", stars: 220000 },
    { name: "vue", stars: 205000 },

  ]}
  columns={[
    { key: "name", header: "name" },
    { key: "stars", header: "stars" },
  ]}
  striped
/>

### Compact

Reduced padding:

```tsx
<Table data={data} columns={columns} compact />
```

### Column width

Set fixed widths:

```tsx
const columns = [
  { key: "name", header: "name", width: "200px" },
  { key: "stars", header: "stars", width: "100px" },
];
```

## Props

### Table

| prop             | type            | default | description                  |
| ---------------- | --------------- | ------- | ---------------------------- |
| data             | T[]             | -       | Array of row objects         |
| columns          | TableColumn[]   | -       | Column definitions           |
| defaultSort      | string          | -       | Initial sort column key      |
| defaultDirection | "asc" \| "desc" | "asc"   | Initial sort direction       |
| striped          | boolean         | false   | Alternating row colors       |
| hoverable        | boolean         | true    | Row hover effect             |
| compact          | boolean         | false   | Reduced padding              |
| caption          | string          | -       | Accessible table description |
| className        | string          | -       | Additional CSS classes       |

### Tablecolumn

| prop         | type     | default | description             |
| ------------ | -------- | ------- | ----------------------- |
| key          | string   | -       | Property name from data |
| header       | string   | -       | Column header text      |
| sortable     | boolean  | false   | Enable sorting          |
| width        | string   | -       | Fixed column width      |
| render       | function | -       | Custom cell renderer    |
| hideOnMobile | boolean  | false   | Hide on small screens   |
