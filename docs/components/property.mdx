Document API properties with types, defaults, and descriptions.

## Usage

```mdx
<Properties>
  <Property name="title" type="string" required>
    the page title displayed in the header
  </Property>
  <Property name="description" type="string">
    optional description shown below title
  </Property>
  <Property name="order" type="number" default="0">
    sort order in navigation
  </Property>
</Properties>
```

## Deprecated properties

Mark properties as deprecated:

```mdx
<Property name="oldProp" type="string" deprecated>
  use newProp instead
</Property>
```

<Properties>
  <Property name="oldProp" type="string" deprecated>
    use newProp instead
  </Property>
</Properties>

## Property props

| prop       | type      | default | description          |
| ---------- | --------- | ------- | -------------------- |
| name       | string    | -       | Property name        |
| type       | string    | -       | Property type        |
| required   | boolean   | false   | Mark as required     |
| default    | string    | -       | Default value        |
| deprecated | boolean   | false   | Mark as deprecated   |
| children   | ReactNode | -       | Property description |

## Properties props

| prop     | type      | default | description    |
| -------- | --------- | ------- | -------------- |
| children | ReactNode | -       | Property items |
