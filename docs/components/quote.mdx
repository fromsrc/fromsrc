## quote

```mdx
<Quote author="Jane Smith" role="CEO at Acme">
  this framework changed how we build documentation
</Quote>
```

## testimonials grid

display multiple testimonials in a grid:

```mdx
<Testimonials>
  <Testimonial author="Alice" role="Engineer">
    fast builds, great dx
  </Testimonial>
  <Testimonial author="Bob" role="Designer">
    beautiful default styles
  </Testimonial>
</Testimonials>
```

## props

### quote

| prop | type | default | description |
|------|------|---------|-------------|
| author | string | - | author name |
| role | string | - | author role |
| avatar | string | - | avatar url |
| children | ReactNode | - | quote text |

### testimonials

| prop | type | default | description |
|------|------|---------|-------------|
| children | ReactNode | - | testimonial items |

### testimonial

| prop | type | default | description |
|------|------|---------|-------------|
| author | string | required | author name |
| role | string | - | author role |
| avatar | string | - | avatar url |
| children | ReactNode | - | quote text |
