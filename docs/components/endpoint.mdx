document rest api endpoints with method badges and parameters.

## usage

```mdx
<Endpoint method="GET" path="/api/users/:id" description="get a user by id">
  <Param name="id" type="string" required description="the user id" />
</Endpoint>
```

<Endpoint method="GET" path="/api/users/:id" description="get a user by id">
  <Param name="id" type="string" required description="the user id" />
</Endpoint>

## methods

all http methods are supported with color coding:

<Endpoint method="GET" path="/api/resource" />
<Endpoint method="POST" path="/api/resource" />
<Endpoint method="PUT" path="/api/resource/:id" />
<Endpoint method="PATCH" path="/api/resource/:id" />
<Endpoint method="DELETE" path="/api/resource/:id" />

## parameters

```mdx
<Endpoint method="POST" path="/api/users" description="create a new user">
  <Param name="name" type="string" required description="user's full name" />
  <Param name="email" type="string" required description="user's email address" />
  <Param name="role" type="string" description="user role (admin, user)" />
</Endpoint>
```

## responses

document response codes:

```mdx
<Response status={200} description="success">
  returns the user object
</Response>

<Response status={404} description="not found">
  user does not exist
</Response>
```

## props

### endpoint

| prop | type | default | description |
|------|------|---------|-------------|
| method | string | required | http method (GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS, TRACE) |
| path | string | required | api path |
| description | string | - | endpoint description |
| children | ReactNode | - | parameter list |

### param

| prop | type | default | description |
|------|------|---------|-------------|
| name | string | required | parameter name |
| type | string | required | parameter type |
| required | boolean | false | marks as required |
| description | string | - | parameter description |

### response

| prop | type | default | description |
|------|------|---------|-------------|
| status | number | required | http status code |
| description | string | - | status description |
| children | ReactNode | - | response details |
