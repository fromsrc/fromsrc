## installation

all components are included in the fromsrc package.

<Tabs items={["bun", "npm"]}>
  <Tab value="bun">
    ```bash
    bun add fromsrc
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install fromsrc
    ```
  </Tab>
</Tabs>

## usage

components are automatically available in your mdx files:

```mdx
<Callout type="info">
  no imports needed.
</Callout>
```
