## Installation

All components are included in the fromsrc package.

<Tabs items={["bun", "npm"]}>
  <Tab value="bun">```bash bun add fromsrc ```</Tab>
  <Tab value="npm">```bash npm install fromsrc ```</Tab>
</Tabs>

## Usage

Components are automatically available in your MDX files:

```mdx
<Callout type="info">no imports needed.</Callout>
```
