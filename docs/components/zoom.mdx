---

wrap images for click-to-zoom functionality.

## Usage

```mdx
<Zoom>![screenshot](/screenshot.png)</Zoom>
```

click the image to zoom, click again or press escape to close.

## With next/image

```mdx
import Image from "next/image";

<Zoom>
  <Image src="/logo.png" width={200} height={100} alt="logo" />
</Zoom>
```

## Props

| prop      | type   | default | description        |
| --------- | ------ | ------- | ------------------ |
| className | string | -       | additional classes |
