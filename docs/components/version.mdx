Dropdown for selecting documentation versions.

## Usage

```tsx
const versions = [
  { id: "v2", label: "v2.0" },
  { id: "v1", label: "v1.0" },
]

<VersionSelect versions={versions} current="v2" />
```

## With navigation

Link to different docs versions:

```tsx
const versions = [
  { id: "v2", label: "v2.0", href: "/v2/docs" },
  { id: "v1", label: "v1.0", href: "/v1/docs" },
]

<VersionSelect versions={versions} current="v2" />
```

## With callback

Handle version changes programmatically:

```tsx
<VersionSelect
  versions={versions}
  current={currentVersion}
  onChange={(version) => setVersion(version)}
/>
```

## Props

| prop     | type      | description                   |
| -------- | --------- | ----------------------------- |
| versions | Version[] | Available versions            |
| current  | string    | Current version ID            |
| onChange | function  | Callback when version changes |

### Version object

| prop  | type   | description             |
| ----- | ------ | ----------------------- |
| id    | string | Version identifier      |
| label | string | Display label           |
| href  | string | Optional navigation URL |
