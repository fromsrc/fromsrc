the steps component creates numbered guides.

## usage

```mdx
<Steps>
### install fromsrc

```bash
bun add fromsrc
```

### create your first doc

add an mdx file to your docs folder.

### run the dev server

```bash
bun dev
```
</Steps>
```

## styling

steps automatically:

- add vertical connector lines
- number each step
- highlight the current step

## nested content

steps can contain any mdx content:

- code blocks
- images
- lists
- other components
