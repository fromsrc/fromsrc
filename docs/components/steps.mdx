the steps component creates numbered guides.

## usage

```mdx
<Steps>
<Step>

### install fromsrc

```bash
bun add fromsrc
```

</Step>
<Step>

### create your first doc

add an mdx file to your docs folder.

</Step>
</Steps>
```

</Step>
</Steps>

## styling

steps automatically:

- add vertical connector lines
- number each step
- support nested content

## nested content

steps can contain any mdx content:

- code blocks
- images
- lists
- other components
