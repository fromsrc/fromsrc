Lightweight toast notifications for user feedback.

## Setup

Wrap your app with the toast provider:

```tsx
import { ToastProvider } from "fromsrc/client";

export default function Layout({ children }) {
  return <ToastProvider>{children}</ToastProvider>;
}
```

## Usage

use the `useToast` hook to trigger notifications:

```tsx
import { useToast } from "fromsrc/client";

function SaveButton() {
  const { add } = useToast();

  return (
    <button
      onClick={() =>
        add({
          type: "success",
          message: "changes saved",
        })
      }
    >
      save
    </button>
  );
}
```

## Types

### Info

```tsx
add({ type: "info", message: "new version available" });
```

### Success

```tsx
add({ type: "success", message: "file uploaded" });
```

### Warning

```tsx
add({ type: "warning", message: "low storage" });
```

### Error

```tsx
add({ type: "error", message: "connection failed" });
```

## Custom duration

Toasts auto-dismiss after 3 seconds by default:

```tsx
add({
  type: "info",
  message: "processing...",
  duration: 5000,
});
```

## Manual dismiss

use `remove` to dismiss a toast early:

```tsx
const { toasts, remove } = useToast();

toasts.map((toast) => (
  <button onClick={() => remove(toast.id)}>dismiss</button>
));
```

## Props

### Toast

| prop     | type                                        | default  | description             |
| -------- | ------------------------------------------- | -------- | ----------------------- |
| message  | string                                      | required | Notification text       |
| type     | "info" \| "success" \| "warning" \| "error" | required | Toast style             |
| duration | number                                      | 3000     | Auto-dismiss time in ms |

### UseToast

| property | type            | description           |
| -------- | --------------- | --------------------- |
| toasts   | Toast[]         | Current active toasts |
| add      | (toast) => void | Show a new toast      |
| remove   | (id) => void    | Dismiss a toast       |
