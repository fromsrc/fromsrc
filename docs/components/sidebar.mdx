Documentation sidebar with search, navigation sections, and collapsible folders.

## Usage

```tsx
import { Sidebar } from "fromsrc/client";

<Sidebar
  title="docs"
  navigation={navigation}
  github="https://github.com/org/repo"
/>;
```

## Navigation structure

### Flat items

Simple list of pages:

```tsx
const navigation = [
  {
    title: "getting started",
    items: docs.filter((d) => d.slug?.startsWith("getting-started")),
  },
];
```

### Collapsible folders

Nested sections that can collapse:

```tsx
const navigation = [
  {
    title: "components",
    items: [
      {
        type: "folder",
        title: "layout",
        items: [
          { type: "item", title: "sidebar", href: "/docs/sidebar" },
          { type: "item", title: "header", href: "/docs/header" },
        ],
      },
      {
        type: "folder",
        title: "content",
        defaultOpen: true,
        items: [
          { type: "item", title: "callout", href: "/docs/callout" },
          { type: "item", title: "steps", href: "/docs/steps" },
        ],
      },
    ],
  },
];
```

### Mixed

Combine doc items with explicit items and folders:

```tsx
const navigation = [
  {
    title: "guides",
    items: [
      ...docs.filter((d) => d.slug?.startsWith("guide")),
      { type: "item", title: "external", href: "https://example.com" },
      {
        type: "folder",
        title: "advanced",
        items: [{ type: "item", title: "plugins", href: "/docs/plugins" }],
      },
    ],
  },
];
```

## Icons

Add icons to items and folders:

```tsx
import { Book, Settings, Puzzle } from "lucide-react";

const navigation = [
  {
    title: "guides",
    items: [
      {
        type: "item",
        title: "intro",
        href: "/docs/intro",
        icon: <Book size={16} />,
      },
      {
        type: "folder",
        title: "config",
        icon: <Settings size={16} />,
        items: [
          {
            type: "item",
            title: "plugins",
            href: "/docs/plugins",
            icon: <Puzzle size={16} />,
          },
        ],
      },
    ],
  },
];
```

Icons appear in the collapsed view and next to items in the expanded view.

## Collapsible sidebar

Enable icon-only collapsed mode with the `collapsible` prop:

```tsx
<Sidebar title="docs" navigation={navigation} collapsible />
```

When collapsed:

- Shows only the toggle button
- Navigation items show icons only
- Expands on hover

## Auto-expand folders

Expand folders automatically to a certain depth:

```tsx
<Sidebar navigation={navigation} defaultOpenLevel={1} />
```

Folders at depth 1 or less will be expanded by default.

## Clickable folders

Folders can also be links:

```tsx
{
  type: "folder",
  title: "api",
  href: "/docs/api",
  items: [
    { type: "item", title: "hooks", href: "/docs/api/hooks" },
    { type: "item", title: "content", href: "/docs/api/content" },
  ]
}
```

Clicking the folder title navigates to the href. The expand/collapse button is separate.

## Props

| prop             | type             | default  | description                  |
| ---------------- | ---------------- | -------- | ---------------------------- |
| title            | string           | required | Site title                   |
| logo             | ReactNode        | -        | Site logo                    |
| navigation       | SidebarSection[] | required | Nav structure                |
| basePath         | string           | "/docs"  | Base URL path                |
| github           | string           | -        | GitHub repo link             |
| collapsible      | boolean          | false    | Enable collapse toggle       |
| defaultOpenLevel | number           | 0        | Auto-expand folders to depth |

## Types

```ts
interface SidebarItem {
  type: "item";
  title: string;
  href: string;
  icon?: ReactNode;
}

interface SidebarFolder {
  type: "folder";
  title: string;
  items: (SidebarItem | SidebarFolder)[];
  defaultOpen?: boolean;
  icon?: ReactNode;
  href?: string;
}

interface SidebarSection {
  title: string;
  items: (SidebarItem | SidebarFolder | DocMeta)[];
}
```
