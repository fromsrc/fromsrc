Render Mermaid diagrams in your docs.

## Installation

Mermaid is a peer dependency:

```bash
bun add mermaid
```

## Usage

```mdx
<Mermaid chart="graph TD; A-->B; B-->C;" />
```

## Examples

### Flowchart

```mdx
<Mermaid
  chart={`
graph TD
    A[start] --> B{decision}
    B -->|yes| C[action]
    B -->|no| D[end]
`}
/>
```

### Sequence diagram

```mdx
<Mermaid
  chart={`
sequenceDiagram
    participant A as client
    participant B as server
    A->>B: request
    B-->>A: response
`}
/>
```

### Entity relationship

```mdx
<Mermaid
  chart={`
erDiagram
    USER ||--o{ POST : writes
    POST ||--|{ COMMENT : has
`}
/>
```

## Props

| prop  | type   | default  | description          |
| ----- | ------ | -------- | -------------------- |
| chart | string | required | Mermaid diagram code |
