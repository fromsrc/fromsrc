Hide content behind expandable sections.

## Usage

```mdx
<Accordion>
  <AccordionItem title="what is fromsrc?">
    a documentation framework for developers.
  </AccordionItem>
  <AccordionItem title="is it free?">
    yes, fromsrc is open source.
  </AccordionItem>
</Accordion>
```

<Accordion>
  <AccordionItem title="What is fromsrc?">
    A documentation framework for developers.
  </AccordionItem>
  <AccordionItem title="Is it free?">
    Yes, fromsrc is open source.
  </AccordionItem>
</Accordion>

## Default open

Start with an item expanded:

```mdx
<AccordionItem title="expanded" defaultOpen>
  visible by default
</AccordionItem>
```

## Props

### AccordionItem

| prop        | type    | default | description    |
| ----------- | ------- | ------- | -------------- |
| title       | string  | -       | Header text    |
| defaultOpen | boolean | false   | Start expanded |
