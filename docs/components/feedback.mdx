the feedback component lets users rate pages as helpful or not helpful.

## Usage

```text
<Feedback onFeedback={(helpful) => console.log(helpful)} />
```

<Feedback />

## Props

| prop       | type                         | description                         |
| ---------- | ---------------------------- | ----------------------------------- |
| onFeedback | `(helpful: boolean) => void` | callback when user submits feedback |

## Tracking feedback

send feedback to your analytics or backend:

```text
<Feedback
  onFeedback={async (helpful) => {
    await fetch('/api/feedback', {
      method: 'POST',
      body: JSON.stringify({ page: '/docs/intro', helpful })
    })
  }}
/>
```
