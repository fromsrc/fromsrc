title: callout
description: highlighted information boxes
order: 2
---

callouts draw attention to important information.

## usage

```mdx
<Callout type="info">
  this is an informational message.
</Callout>
```

## types

### info

default callout for general information.

<Callout type="info">
fromsrc supports mdx out of the box.
</Callout>

### warning

for cautionary messages.

<Callout type="warning">
this action cannot be undone.
</Callout>

### error

for error states or breaking changes.

<Callout type="error">
this api is deprecated in v2.
</Callout>

### tip

for helpful suggestions.

<Callout type="tip">
use keyboard shortcuts for faster navigation.
</Callout>

## with title

add a title to your callout:

<Callout type="info" title="note">
you can add a title to any callout type.
</Callout>

## props

| prop | type | default | description |
|------|------|---------|-------------|
| type | "info" \| "warning" \| "error" \| "tip" | "info" | callout style |
| title | string | - | optional title |
