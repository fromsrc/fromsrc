callouts draw attention to important information.

## Usage

```mdx
<Callout type="info">this is an informational message.</Callout>
```

## Types

### Info

default callout for general information.

<Callout type="info">fromsrc supports mdx out of the box.</Callout>

### Warning

for cautionary messages.

<Callout type="warning">this action cannot be undone.</Callout>

### Error

for error states or breaking changes.

<Callout type="error">this api is deprecated in v2.</Callout>

### Tip

for helpful suggestions.

<Callout type="tip">use keyboard shortcuts for faster navigation.</Callout>

## With title

add a title to your callout:

<Callout type="info" title="note">
  you can add a title to any callout type.
</Callout>

## Props

| prop  | type                                    | default | description    |
| ----- | --------------------------------------- | ------- | -------------- |
| type  | "info" \| "warning" \| "error" \| "tip" | "info"  | callout style  |
| title | string                                  | -       | optional title |
