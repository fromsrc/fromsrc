Breadcrumb shows the current page location.

## Usage

```tsx
import { Breadcrumb } from "fromsrc/client";

<Breadcrumb />;
```

## Base path

Customize the base path:

```tsx
<Breadcrumb base="/documentation" />
```

## Props

| prop | type   | default | description         |
| ---- | ------ | ------- | ------------------- |
| base | string | "/docs" | Base path for links |

## Features

- Auto-generates from current URL
- Converts slugs to readable labels
- Last item is not a link
- Accessible with ARIA label
