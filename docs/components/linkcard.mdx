Styled link cards with optional icons.

## Linkcard

Internal link:

```mdx
<LinkCard
  href="/docs/quickstart"
  title="quickstart"
  description="get started in minutes"
/>
```

<LinkCard
  href="/docs/quickstart"
  title="quickstart"
  description="get started in minutes"
/>

## External link

Shows external icon:

```mdx
<LinkCard
  href="https://github.com"
  title="github"
  description="view source code"
/>
```

<LinkCard
  href="https://github.com"
  title="github"
  description="view source code"
/>

## With icon

```mdx
<LinkCard
  href="/docs/components"
  title="components"
  description="browse all components"
  icon="📦"
/>
```

<LinkCard
  href="/docs/components"
  title="components"

/>

## Link grid

```mdx
<LinkCards>
  <LinkCard href="/docs/install" title="install" />
  <LinkCard href="/docs/config" title="config" />
</LinkCards>
```

<LinkCards>
  <LinkCard href="/docs/quickstart" title="quickstart" />
  <LinkCard href="/docs/customization" title="customization" />
</LinkCards>

## Props

### Linkcard

| prop        | type      | default | description      |
| ----------- | --------- | ------- | ---------------- |
| href        | string    | -       | Link URL         |
| title       | string    | -       | Link title       |
| description | string    | -       | Link description |
| icon        | ReactNode | -       | Optional icon    |
