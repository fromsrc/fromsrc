embed content from external platforms.

## youtube

```mdx
<YouTube id="dQw4w9WgXcQ" title="demo video" />
```

## codesandbox

```mdx
<CodeSandbox id="new" title="react example" />
```

## stackblitz

```mdx
<StackBlitz id="vitejs-vite-template" file="src/App.tsx" />
```

## tweet

```mdx
<Tweet id="1234567890" />
```

## gist

```mdx
<Gist id="username/gist-id" file="example.js" />
```

## props

### youtube

| prop | type | default | description |
|------|------|---------|-------------|
| id | string | - | youtube video id |
| title | string | "YouTube video" | iframe title |

### codesandbox

| prop | type | default | description |
|------|------|---------|-------------|
| id | string | - | sandbox id |
| title | string | "CodeSandbox" | iframe title |

### stackblitz

| prop | type | default | description |
|------|------|---------|-------------|
| id | string | - | project id |
| file | string | - | file to open |
| title | string | "StackBlitz" | iframe title |

### tweet

| prop | type | default | description |
|------|------|---------|-------------|
| id | string | - | tweet id |

### gist

| prop | type | default | description |
|------|------|---------|-------------|
| id | string | - | gist id (user/id) |
| file | string | - | specific file |
