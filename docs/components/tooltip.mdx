Small popover that appears on hover.

## Usage

```mdx
<Tooltip content="additional info here">hover me</Tooltip>
```

<Tooltip content="this is a tooltip">hover over this text</Tooltip>

## Position

### Top

Default position above the element.

<Tooltip content="above" side="top">
  hover (top)
</Tooltip>

### Bottom

Position below the element.

<Tooltip content="below" side="bottom">
  hover (bottom)
</Tooltip>

## Delay

Control how long before the tooltip appears.

```mdx
<Tooltip content="quick" delay={0}>
  instant
</Tooltip>
<Tooltip content="slow" delay={500}>
  half second
</Tooltip>
```

<Tooltip content="instant tooltip" delay={0}>
  instant
</Tooltip>
·
<Tooltip content="delayed tooltip" delay={500}>
  half second delay
</Tooltip>

## Props

| prop     | type              | default  | description       |
| -------- | ----------------- | -------- | ----------------- |
| content  | ReactNode         | required | Tooltip content   |
| children | ReactNode         | required | Trigger element   |
| side     | "top" \| "bottom" | "top"    | Position          |
| delay    | number            | 200      | Ms before showing |
