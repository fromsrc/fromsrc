Form components for collecting user input.

## Input

Text input with label, error, and hint support.

### Usage

```tsx
import { Input } from "fromsrc/client";

<Input label="email" placeholder="you@example.com" />;
```

<Input label="email" placeholder="you@example.com" />

### Sizes

<div className="flex flex-col gap-4">
  <Input size="sm" placeholder="small" />
  <Input size="md" placeholder="medium" />
  <Input size="lg" placeholder="large" />
</div>

### With error

<Input label="username" error="username is required" />

### With hint

<Input label="password" type="password" hint="must be 8+ characters" />

### With tooltip

<Input label="API key" tooltip="Find this in your dashboard settings" />

### Props

| prop    | type                 | default   | description              |
| ------- | -------------------- | --------- | ------------------------ |
| variant | "default" \| "error" | "default" | Visual style             |
| size    | "sm" \| "md" \| "lg" | "md"      | Input size               |
| label   | ReactNode            | -         | Accessible label         |
| error   | ReactNode            | -         | Error message            |
| hint    | ReactNode            | -         | Helper text              |
| tooltip | ReactNode            | -         | Help text shown on hover |

## Autofill

Search input with autocomplete suggestions.

### Usage

```tsx
import { Autofill } from "fromsrc/client";

<Autofill
  items={[
    { id: "1", label: "javascript" },
    { id: "2", label: "typescript" },
    { id: "3", label: "python" },
  ]}
  placeholder="search languages"
/>;
```

<Autofill
  items={[
    { id: "1", label: "javascript" },
    { id: "2", label: "typescript" },
    { id: "3", label: "python" },
    { id: "4", label: "rust" },
    { id: "5", label: "go" },
  ]}
  placeholder="search languages"
/>

### With selection callback

```tsx
<Autofill items={items} onSelect={(item) => console.log("selected:", item)} />
```

### Features

- Filters as you type
- Keyboard navigation with arrow keys
- `Enter` to select highlighted item
- `Escape` to close suggestions
- Closes on outside click

### Props

| prop        | type                         | default     | description           |
| ----------- | ---------------------------- | ----------- | --------------------- |
| items       | AutofillItem[]               | required    | List of suggestions   |
| placeholder | string                       | "search..." | Placeholder text      |
| value       | string                       | -           | Controlled value      |
| onChange    | (value: string) => void      | -           | Input change callback |
| onSelect    | (item: AutofillItem) => void | -           | Selection callback    |

### AutofillItem

| prop  | type   | default  | description                             |
| ----- | ------ | -------- | --------------------------------------- |
| id    | string | required | Unique identifier                       |
| label | string | required | Display text                            |
| value | string | -        | Value when selected (defaults to label) |

## Radio

Radio button group for selecting one option from a set.

### Usage

```tsx
import { Radio, RadioGroup } from "fromsrc/client";

<RadioGroup label="plan">
  <Radio value="free" label="free" />
  <Radio value="pro" label="pro" />
  <Radio value="team" label="team" />
</RadioGroup>;
```

<RadioGroup label="plan" defaultValue="pro">
  <Radio value="free" label="free" />
  <Radio value="pro" label="pro" />
  <Radio value="team" label="team" />
</RadioGroup>
