small labels for status indicators.

## Usage

```mdx
<Badge>default</Badge>
<Badge variant="success">stable</Badge>
```

<Badge>default</Badge> <Badge variant="success">stable</Badge>
<Badge variant="warning">beta</Badge> <Badge variant="error">deprecated</Badge>
<Badge variant="info">new</Badge>

## Variants

### Default

neutral style for general labels.

<Badge>v1.0.0</Badge>

### Success

for positive status.

<Badge variant="success">stable</Badge>

### Warning

for cautionary status.

<Badge variant="warning">beta</Badge>

### Error

for negative status.

<Badge variant="error">deprecated</Badge>

### Info

for informational status.

<Badge variant="info">new</Badge>

## Props

| prop    | type                                                     | default   | description |
| ------- | -------------------------------------------------------- | --------- | ----------- |
| variant | "default" \| "success" \| "warning" \| "error" \| "info" | "default" | badge style |
