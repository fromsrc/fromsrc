Toggle content visibility with smooth animations.

## Collapsible

Controlled component with state:

```mdx
<Collapsible title="Show more details">
  Hidden content that appears when expanded. Supports any content including code
  blocks and components.
</Collapsible>
```

<Collapsible title="Show more details">
  Hidden content that appears when expanded. Supports any content including code
  blocks and components.
</Collapsible>

## Default open

Start expanded:

```mdx
<Collapsible title="Expanded by default" defaultOpen>
  This content is visible on page load
</Collapsible>
```

<Collapsible title="Expanded by default" defaultOpen>

</Collapsible>

## Details

Native HTML details element with styled appearance:

```mdx
<Details summary="Click to expand">

</Details>
```

| component   | use case                            |
