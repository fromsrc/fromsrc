Text that can be copied with one click.

## Inline copyable

```mdx
<Copyable value="npm install fromsrc" />
```

<Copyable value="npm install fromsrc" />

## With label

```mdx
<Copyable label="install:" value="npm install fromsrc" />
```

<Copyable label="install:" value="npm install fromsrc" />

## Copy block

Full-width copyable block:

```mdx
<CopyBlock>NEXT_PUBLIC_API_KEY=your-api-key</CopyBlock>
```

<CopyBlock>NEXT_PUBLIC_API_KEY=your-api-key</CopyBlock>

## Props

### Copyable

| prop  | type   | default | description    |
| ----- | ------ | ------- | -------------- |
| value | string | -       | Text to copy   |
| label | string | -       | Optional label |

### CopyBlock

| prop     | type   | default | description  |
| -------- | ------ | ------- | ------------ |
| children | string | -       | Text to copy |
