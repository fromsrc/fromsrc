### Create your first page

create `docs/index.mdx`:

```mdx
---
title: welcome
description: my documentation
---

# welcome

this is your documentation home page.
```

</Step>
<Step>

### Start the dev server

```bash
bun run dev:up
bun run dev:status
```

open [http://localhost:3000/docs](http://localhost:3000/docs) to see your site.

when you are done:

```bash
bun run dev:down
```

</Step>
</Steps>

## Add more pages

create `docs/getting-started.mdx`:

```mdx
---
title: getting started
description: how to get started

follow these steps to set up the project.

## Install dependencies

<Install package="your-package" />

## Configure

create a config file and add your settings.
```

## Customize the sidebar

pages are sorted by the `order` field in frontmatter:

```yaml
---
title: page title
order: 1
---
```

lower numbers appear first. organize with folders:

<Files>
  <Folder name="docs" defaultOpen>
    <File name="index.mdx" />
    <Folder name="guides" defaultOpen>
      <File name="index.mdx" />
      <File name="setup.mdx" />
    </Folder>
    <Folder name="api" defaultOpen>
      <File name="index.mdx" />
      <File name="endpoints.mdx" />
    </Folder>
  </Folder>
</Files>

folders become collapsible sections in the sidebar.
