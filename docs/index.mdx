fromsrc is a documentation framework built for performance and full control.

## why fromsrc

we built fromsrc around four principles:

- **composable** - small pieces that combine freely
- **editable** - change anything without fighting the framework
- **customizable** - swap any component, style, or behavior
- **optimized** - incremental builds, scales to thousands of files

## quick start

```bash
bunx create-fromsrc
cd my-docs
bun dev
```
