## features

- **mdx** - write docs with react components embedded
- **search** - fast fuzzy search with cmd+k and keyboard hints
- **keyboard nav** - arrow keys navigate between pages
- **collapsible sidebar** - icon-only mode for more space
- **dark mode** - built-in theme support
- **responsive** - mobile-friendly layout
- **llms.txt** - ai-ready content export
- **composable** - every component is customizable
