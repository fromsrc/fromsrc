## Get started

<Steps>
  <Step>

### Install

<Install package="fromsrc" />

  </Step>
  <Step>

### Create your first doc

Add an MDX file to your docs folder.

```mdx title="docs/index.mdx"
---
title: welcome
---

your documentation starts here.
```

  </Step>
  <Step>

### Run dev server

```bash
bun run dev:up
bun run dev:status
```

  </Step>
</Steps>
