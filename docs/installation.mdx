## create a new project

the fastest way to start is with create-fromsrc:

<Create />

this will:

1. create a new directory
2. set up next.js with fromsrc
3. add example documentation
4. install dependencies

## manual installation

add fromsrc to an existing next.js project:

<Install package="fromsrc" />

## project structure

```
my-docs/
├── app/
│   └── docs/
│       └── [[...slug]]/
│           └── page.tsx
├── docs/
│   ├── index.mdx
│   └── getting-started.mdx
└── package.json
```
