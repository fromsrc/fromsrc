## create a new project

the fastest way to start is with create-fromsrc:

<Create />

<Steps>
<Step>

### run the command

this creates a new directory with everything configured.

</Step>
<Step>

### enter the directory

```bash
cd my-docs
```

</Step>
<Step>

### start the dev server

```bash
bun dev
```

</Step>
</Steps>

## manual installation

add fromsrc to an existing next.js project:

<Install package="fromsrc" />

<Callout type="info">
fromsrc requires next.js 16 or later.
</Callout>
