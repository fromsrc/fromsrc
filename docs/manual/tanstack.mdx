---
title: TanStack Start
description: Configure fromsrc in TanStack Start apps
---

Install dependencies:

```bash
bun add fromsrc @tanstack/react-router
```

Wire the adapter:

```tsx
import { AdapterProvider, tanstackAdapter } from "fromsrc/tanstack";
import type { ReactNode } from "react";

export function Providers({ children }: { children: ReactNode }) {
  return (
    <AdapterProvider adapter={tanstackAdapter}>{children}</AdapterProvider>
  );
}
```

Use the provider near your router provider so link/navigation components are consistent.

Use the core content helpers from `fromsrc` for docs parsing and search indexing.
