## fromsrc.config.ts

Create a config file in your project root:

```typescript
import { defineConfig } from "fromsrc";

export default defineConfig({
  title: "my docs",
  description: "documentation for my project",

  navigation: {
    auto: true,
  },

  search: {
    enabled: true,
  },

  theme: {
    defaultTheme: "dark",
  },
});
```

## Options

### Title

The site title shown in the header and metadata.

```typescript
{
  title: "my docs";
}
```

### Description

Default meta description for pages without one.

```typescript
{
  description: "documentation for my project";
}
```

### Navigation

Configure sidebar navigation:

<Accordion>
<AccordionItem title="auto navigation">

Generate navigation from your file structure automatically.

```typescript
{
  navigation: {
    auto: true;
  }
}
```

</AccordionItem>
<AccordionItem title="manual navigation">

Define navigation items manually for full control.

```typescript
{
  navigation: {
    items: [
      { title: "introduction", href: "/docs" },
      { title: "installation", href: "/docs/installation" },
    ];
  }
}
```

</AccordionItem>
</Accordion>

### Search

Configure search behavior:

| option      | type    | description              |
| ----------- | ------- | ------------------------ |
| enabled     | boolean | enable/disable search    |
| placeholder | string  | search input placeholder |

### Theme

Configure theming:

| option       | type              | description          |
| ------------ | ----------------- | -------------------- |
| defaultTheme | "dark" \| "light" | default color scheme |

<Callout type="tip">fromsrc uses dark mode by default.</Callout>
